# 家庭菜谱看板

记录家的味道的可视化菜谱查看工具。

## 功能特点

- 🍳 **菜谱浏览** - 网格卡片展示，图片 + 菜名 + 标签
- 📖 **详情查看** - 点击卡片查看完整原料和做法
- 📱 **响应式** - 桌面手机都能用，手机端两列展示
- 🎨 **自然风格** - 绿色系配色，流动渐变背景
- 🖼️ **本地图片** - 支持本地图片，无外部依赖

## 快速开始

### 在线访问

直接打开：https://yiqianfan.github.io/Recipes-Board/

### 本地使用

1. 下载 `index.html` 和图片文件
2. 用浏览器打开 `index.html` 即可使用

数据自动保存在浏览器本地（localStorage），无需注册账号。

## 使用说明

### 浏览菜谱
首页即是菜谱网格，点击任意卡片查看详情。

### 图片说明
- **有本地图片的菜品**：显示真实菜品照片
- **无本地图片的菜品**：显示渐变背景 + 菜名占位符
- 如需添加图片，可通过 AI 修改代码中的 `sampleRecipes` 数组

## 技术栈

- HTML5 + CSS3 + JavaScript
- 无框架、无依赖
- localStorage 数据持久化

## 内容管理

更新菜谱内容通过 AI 修改 `index.html` 中的 `sampleRecipes` 数组即可，格式：

```javascript
{
  id: 1,
  name: "菜名",
  image: "./图片路径.jpg",  // 本地图片或网络URL，留空则显示占位符
  tags: ["标签1", "标签2"],
  ingredients: ["原料1 用量", "原料2 用量"],
  steps: ["步骤1", "步骤2"],
  createdAt: new Date().toISOString()
}
```
