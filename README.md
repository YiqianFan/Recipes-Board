# 家庭菜谱看板

记录家的味道的可视化菜谱管理工具。

![家庭菜谱看板](https://images.unsplash.com/photo-1556910103-1c02745aae4d?w=800&h=400&fit=crop)

## 功能特点

- 🍳 **菜谱浏览** - 网格卡片展示，图片 + 菜名 + 标签
- 📖 **详情查看** - 点击卡片查看完整原料和做法
- ➕ **新增/编辑** - 表单轻松添加新菜谱
- 📱 **响应式** - 桌面手机都能用，手机端两列展示
- 🎨 **自然风格** - 绿色系配色，流动渐变背景

## 快速开始

### 在线访问

直接打开：https://yiqianfan.github.io/Recipes-Board/

### 本地使用

1. 下载 `index.html`
2. 用浏览器打开即可使用

数据自动保存在浏览器本地（localStorage），无需注册账号。

## 使用说明

### 浏览菜谱
首页即是菜谱网格，点击任意卡片查看详情。

### 添加菜谱
1. 点击右下角 **+** 按钮
2. 填写菜名、图片URL、标签
3. 填写原料（每行一个）和做法（每行一步）
4. 点击保存

### 编辑/删除
1. 点击右上角 **⚙️ 管理**
2. 找到要编辑的菜谱，点 **编辑** 或 **删除**

### 图片
目前支持两种方式：
- **网络图片**：粘贴图片 URL
- **本地图片**：放在项目文件夹，用相对路径引用（如 `./pic/dish.jpg`）

暂不支持本地上传功能。
